**Model Context Protocol (MCP)**를 활용하여 날씨 예보 서버(weather forecast server)를 구축하는 과정에서 사용할 수 있는 단계별 프롬프트를 아래에 정리하겠습니다. 이 프롬프트는 MCP 서버 개발 가이드와 Claude Desktop 클라이언트를 기반으로 하며, 실제로 사용 가능한 형태로 작성되었습니다.
