AI로 유튜브 분석 도구 만들기: 월 300만원짜리 서비스를 무료로!

유튜브 크리에이터들에게 필수적인 도구인 '뷰트랩(VidIQ)'은 월 300만원이라는 높은 비용으로 인해 많은 이들에게 부담이 되고 있습니다. 하지만 AI의 도움을 받아 이와 유사한 기능을 무료로 구현할 수 있다는 사실, 알고 계셨나요? 이번 글에서는 AI와 구글 스프레드시트의 앱스 스크립트를 활용해 뷰트랩과 유사한 유튜브 분석 도구를 만드는 방법을 상세히 알아보겠습니다.

뷰트랩(VidIQ) 소개

뷰트랩은 유튜브 영상을 분석해주는 강력한 서비스입니다. 주요 기능으로는:

  • 키워드 검색을 통한 영상 분석
  • 조회수, 구독자 수, 영상 수 등의 데이터 제공
  • 노출 확률, 기호도, 성과도 등의 지표 제공
  • 채널 분석 기능

하지만 이러한 기능을 모두 사용하려면 월 300만원이라는 높은 비용을 지불해야 합니다.

AI를 활용한 무료 대안 만들기

AI의 도움을 받아 구글 스프레드시트와 앱스 스크립트를 활용하면, 뷰트랩과 유사한 기능을 무료로 구현할 수 있습니다. 이 방법을 통해 다음과 같은 데이터를 얻을 수 있습니다:

  • 유튜브 영상 ID
  • 영상 제목
  • 채널명
  • 영상 설명
  • 업로드 날짜
  • 썸네일 URL
  • 조회수
  • 구독자 수
  • 총 영상 수
  • 성과도 (구독자 수 대비 조회수)
  • 최신 성과도 (업로드 날짜 고려)

구현 과정

  1. 구글 클라우드 콘솔 설정
    • 구글 클라우드 콘솔에 접속
    • 새 프로젝트 생성 (예: 유튜브 애널리틱스)
    • YouTube Data API v3 활성화
    • API 키 발급
  2. 구글 스프레드시트 생성
    • 새 스프레드시트 생성
    • 도구 > 스크립트 편집기 선택
  3. 앱스 스크립트 코드 작성
    • AI에게 코드 요청
    • 제공받은 코드를 스크립트 편집기에 붙여넣기
    • API 키 설정 함수 실행
  4. 데이터 가져오기
    • 키워드 입력 후 실행
    • 결과 확인 (영상 ID, 제목, 채널명 등)
  5. 추가 데이터 요청
    • 조회수, 구독자 수, 총 영상 수 추가 요청
    • 코드 수정 및 재실행
  6. 성과도 계산
    • 구독자 수 대비 조회수로 성과도 계산
    • 코드 수정 및 재실행
  7. UI 개선
    • 메뉴 형식으로 API 키 설정 및 검색 기능 구현
    • 코드 수정 및 재실행
  8. 최신 성과도 추가
    • 업로드 날짜를 고려한 최신 성과도 계산
    • 코드 수정 및 재실행

주요 기능 사용 방법 (20단계)

  1. 구글 클라우드 콘솔 접속
  2. 새 프로젝트 생성 (유튜브 애널리틱스)
  3. YouTube Data API v3 검색
  4. API 활성화
  5. 사용자 인증 정보 생성
  6. API 키 복사
  7. 구글 스프레드시트 새로 만들기
  8. 도구 메뉴에서 스크립트 편집기 선택
  9. AI에게 코드 요청
  10. 제공받은 코드를 스크립트 편집기에 붙여넣기
  11. 코드 저장
  12. 실행 버튼 옆 드롭다운 메뉴에서 'setApiKey' 함수 선택
  13. 실행 버튼 클릭
  14. 복사한 API 키 입력
  15. 'getYoutubeSearchResults' 함수 선택
  16. 실행 버튼 클릭
  17. 검색할 키워드 입력 (예: 마케팅)
  18. 결과 확인 (스프레드시트에 데이터 자동 입력)
  19. 메뉴에서 '유튜브 검색' 선택
  20. '유튜브 검색 실행' 클릭 후 키워드 입력하여 새로운 검색 수행

결과 분석

이렇게 만든 도구를 통해 다음과 같은 분석이 가능합니다:

  1. 일반 성과도: 구독자 수 대비 조회수가 높은 영상 파악
  2. 최신 성과도: 최근 업로드된 영상 중 성과가 좋은 콘텐츠 식별
  3. 트렌드 파악: 특정 키워드에 대한 인기 있는 최신 콘텐츠 확인
  4. 경쟁 분석: 경쟁 채널의 성과 비교 분석
  5. 콘텐츠 전략 수립: 성공적인 영상의 특징 파악 및 적용
