PPT에 활용할 수 있는 무료 AI 도구 활용 가이드

개요

이 글은 디자이너가 아닌 일반 직장인들이 AI를 활용하여 보고서나 PPT에 들어갈 이미지와 디자인 요소를 만드는 방법을 소개합니다. 실무에 바로 적용할 수 있는 포인트 이미지, 아이콘, 픽토그램, 다이어그램, 플로우 차트 등을 만드는 데 초점을 맞추고 있습니다. 소개되는 도구들은 무료로 사용 가능하며, 사용하기 쉬운 것들로 선별되었습니다.

주요 AI 도구 소개

1. Recraft (리크래프트)

Recraft는 무료로 사용할 수 있는 AI 이미지 생성 도구 중 가장 우수한 도구입니다.

주요 특징:

  • 무료 사용자도 하루 25개의 이미지 생성 가능
  • 다양한 프리셋 이미지 스타일 제공
  • 벡터 이미지(SVG) 생성 기능

사용 팁:

  • 한글로 프롬프트 작성 가능
  • 실사보다는 일러스트나 아이콘 스타일 이미지 생성에 더 적합
  • SVG 형식으로 다운로드하여 PowerPoint에서 편집 가능

2. Napkin AI (냅킨 AI)

Napkin AI는 보고서에 주로 사용되는 차트, 다이어그램, 구조도, 플로우 차트 등을 무료로 만들 수 있는 도구입니다.

주요 특징:

  • 텍스트 입력만으로 다양한 시각화 요소 생성
  • 다양한 템플릿 제공
  • SVG 형식으로 다운로드 가능

사용 팁:

  • 구조화된 형태로 내용 입력 시 더 좋은 결과물 생성
  • ChatGPT를 활용하여 내용 구조화 및 시각화 아이디어 얻기

3. Claude

Claude는 ChatGPT와 유사한 대규모 언어 모델로, 시각화 목적으로도 활용 가능합니다.

주요 특징:

  • React 기반의 웹 개발 언어를 사용한 시각화
  • 차트뿐만 아니라 플로우 차트, 다이어그램 등 다양한 시각화 가능

사용 팁:

  • 구체적인 프롬프트 작성으로 원하는 결과물 얻기
  • SVG 형식으로 변환하여 PowerPoint에서 편집 가능

4. 미리캔버스

미리캔버스는 한국의 비디자이너를 위한 디자인 툴로, 최근 AI 프레젠테이션 기능이 추가되었습니다.

주요 특징:

  • 한국인의 취향에 맞는 디자인 템플릿 제공
  • AI를 활용한 프레젠테이션 자동 생성
  • PowerPoint 파일로 다운로드 가능

사용 팁:

  • ChatGPT로 구조화된 내용을 만들어 입력하면 더 좋은 결과물 얻기 가능
  • '딱 맞는 페이지 찾기' 기능을 활용하여 템플릿 선택

AI 도구 활용 30단계 가이드

  1. 프로젝트의 목적과 대상 청중 파악하기
  2. 필요한 시각 자료 유형 결정하기 (차트, 다이어그램, 아이콘 등)
  3. Recraft에서 적합한 이미지 스타일 선택하기
  4. 한글로 프롬프트 작성하여 Recraft에서 이미지 생성하기
  5. 생성된 이미지 중 가장 적합한 것 선택하기
  6. 필요시 Recraft의 이미지 세트 기능을 활용하여 일관된 스타일의 여러 이미지 생성하기
  7. SVG 형식으로 이미지 다운로드하기
  8. PowerPoint에 SVG 이미지 삽입하기
  9. 삽입된 이미지 그룹 해제하여 개별 요소 편집하기
  10. Napkin AI에서 보고서 내용 입력하기
  11. ChatGPT를 활용하여 내용 구조화하기
  12. Napkin AI에서 시각화 버튼 클릭하여 다양한 시각화 옵션 확인하기
  13. 가장 적합한 시각화 옵션 선택하기
  14. 필요시 Napkin AI에서 텍스트 및 아이콘 수정하기
  15. Napkin AI에서 생성된 시각 자료 SVG 형식으로 다운로드하기
  16. Claude에서 시각화할 내용 입력하기
  17. Claude가 제안한 시각화 계획 검토하기
  18. Claude가 생성한 React 코드 기반 시각화 결과물 확인하기
  19. 필요시 Claude에 피드백을 제공하여 결과물 개선하기
  20. Claude의 결과물을 SVG 형식으로 변환하기
  21. 미리캔버스에서 AI 프레젠테이션 만들기 메뉴 선택하기
  22. ChatGPT로 구조화한 내용을 미리캔버스에 입력하기
  23. 미리캔버스가 제안한 프레젠테이션 개요 검토 및 수정하기
  24. 적합한 디자인 템플릿 선택하기
  25. 미리캔버스가 생성한 프레젠테이션 초안 검토하기
  26. '딱 맞는 페이지 찾기' 기능을 활용하여 추가 슬라이드 삽입 또는 수정하기
  27. Recraft, Napkin AI, Claude에서 생성한 시각 자료들을 미리캔버스 프레젠테이션에 통합하기
  28. 전체적인 디자인 일관성 확인 및 조정하기
  29. 최종 프레젠테이션 검토 및 미세 조정하기
  30. 완성된 프레젠테이션 PowerPoint 파일로 다운로드하기

이러한 AI 도구들을 활용하면 디자인 전문가가 아니더라도 효율적이고 전문적으로 보이는 프레젠테이션 자료를 만들 수 있습니다. 각 도구의 장점을 조합하여 사용하면 더욱 효과적인 결과물을 얻을 수 있습니다. AI는 우리의 작업을 보조하는 도구일 뿐이며, 최종적인 판단과 조정은 사용자의 몫임을 기억하세요.
