Windsurf 소개

Windsurf는 Codium이라는 회사에서 개발한 AI 기반 IDE입니다. 원래 VS Code의 코드 자동완성 확장 프로그램으로 시작했지만, 이제는 AI 에이전트와 코파일럿을 활용한 강력한 개발 도구로 발전했습니다.

Windsurf의 주요 특징:

  • 코드베이스에 대한 깊은 이해
  • 실시간 사용자 액션 인식
  • LLM 기반 코드 검색
  • 이미지 업로드 및 UI 생성 기능

프로젝트 1: 인보이스 분석기

첫 번째 프로젝트로 인보이스 분석기를 만들어보겠습니다. 이 앱의 주요 기능은 다음과 같습니다:

  • PDF 인보이스 업로드
  • 추출할 필드 선택
  • Azure Document Intelligence를 사용한 데이터 추출
  • JSON/CSV 형식으로 결과 다운로드

개발 단계

  1. Azure 계정 설정
  2. Document Intelligence 서비스 생성
  3. Next.js 프로젝트 생성
  4. Windsurf에서 instructions.md 파일 작성
  5. 기능별 개발 및 테스트
  6. UI 개선
  7. 배포 준비

프로젝트 2: 콘텐츠 재활용 앱

두 번째 프로젝트는 소셜 미디어 콘텐츠를 생성하는 앱입니다. 주요 기능:

  • 사용자 인증
  • 기사 생성
  • LinkedIn/Twitter 포스트 생성
  • 토큰 기반 사용량 제한
  • Stripe 결제 통합

개발 단계

  1. Supabase 설정
  2. 사용자 인증 구현
  3. 페르소나 관리 기능
  4. 기사 생성 기능
  5. 소셜 미디어 포스트 생성 기능
  6. 토큰 관리 시스템
  7. Stripe 결제 통합
  8. UI 최적화 및 반응형 디자인

프로젝트 3: 실시간 번역 메신저

마지막 프로젝트는 실시간 번역 기능이 있는 메신저 앱입니다. 주요 기능:

  • 사용자 인증
  • 프로필 관리 (언어 설정 등)
  • 연락처 검색 및 추가
  • 실시간 메시징
  • OpenAI를 이용한 실시간 번역

개발 단계

  1. Next.js 프로젝트 설정
  2. Supabase 연동
  3. 사용자 인증 구현
  4. 프로필 관리 페이지 개발
