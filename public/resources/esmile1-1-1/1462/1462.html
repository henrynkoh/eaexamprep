유튜브 채널 운영자라면 VidIQ와 같은 분석 도구가 얼마나 유용한지 잘 아실 겁니다. 하지만 구독료가 부담스러울 수도 있죠. 이 글에서는 구글 시트, 구글 앱스 스크립트, 유튜브 API를 활용하여 VidIQ와 유사한 기능을 직접 구축하는 방법을 자세히 알아보겠습니다. 30단계로 세분화된 상세 가이드를 통해, 코딩 경험이 없는 분들도 쉽게 따라 할 수 있도록 안내해 드리겠습니다.

 

왜 구글 시트, 앱스 스크립트, 유튜브 API를 사용해야 할까요?

  • 무료: 구글 시트와 앱스 스크립트는 기본적으로 무료로 제공됩니다. 유튜브 API는 일정한 할당량 내에서 무료로 사용할 수 있습니다.
  • 유연성: 직접 코드를 작성하여 원하는 기능을 자유롭게 추가하고 수정할 수 있습니다.
  • 자동화: 앱스 스크립트를 사용하여 데이터 수집 및 분석을 자동화할 수 있습니다.
  • 데이터 제어: 모든 데이터는 구글 시트에 저장되므로, 데이터에 대한 완전한 제어권을 가질 수 있습니다.
  • 학습 기회: 앱스 스크립트와 유튜브 API를 사용하면서 프로그래밍 및 데이터 분석 능력을 향상시킬 수 있습니다.

기본 개념 이해

  • 구글 시트: 데이터를 저장하고 관리하는 스프레드시트 도구입니다.
  • 구글 앱스 스크립트: 구글 제품(시트, 문서, 드라이브 등)을 자동화하고 확장하는 데 사용되는 스크립트 언어입니다.
  • 유튜브 API: 유튜브 데이터에 접근하고 조작할 수 있도록 제공되는 인터페이스입니다.

30단계 상세 가이드

 

1단계: 구글 시트 생성 및 설정

  1. 구글 드라이브에 접속하여 새 구글 시트를 만듭니다.
  2. 시트 이름을 '유튜브 분석'과 같이 적절하게 변경합니다.
  3. 시트 상단에 열 제목을 추가합니다. (예: 동영상 ID, 제목, 조회수, 좋아요, 댓글수, 게시일, 태그, 설명, 채널 ID, 채널 이름, 구독자 수 등).

2단계: 유튜브 API 키 획득

  1. 구글 개발자 콘솔에 접속합니다.
  2. 새 프로젝트를 만들거나 기존 프로젝트를 선택합니다.
  3. 'API 및 서비스' > '라이브러리'로 이동하여 'YouTube Data API v3'를 검색하고 활성화합니다.
  4. '사용자 인증 정보' 페이지에서 '사용자 인증 정보 만들기' > 'API 키'를 선택하여 API 키를 생성합니다.
  5. API 키를 안전한 곳에 저장합니다. 이 키는 앱스 스크립트에서 사용됩니다.

3단계: 앱스 스크립트 편집기 열기

  1. 구글 시트에서 '확장 프로그램' > 'Apps Script'를 클릭합니다.
  2. 새 스크립트 편집기가 열립니다.
  3. 프로젝트 이름을 '유튜브 분석'과 같이 적절하게 변경합니다.

function getVideoInfo(videoId) { try { var response = YouTube.Videos.list('snippet,statistics,contentDetails', { id: videoId, key: API_KEY }); if (response.items.length === 0) { return null; // 동영상 정보가 없는 경우 null 반환 } var video = response.items[0]; return { title: video.snippet.title, viewCount: video.statistics.viewCount, likeCount: video.statistics.likeCount, commentCount: video.statistics.commentCount, publishedAt: video.snippet.publishedAt, tags: video.snippet.tags, description: video.snippet.description, channelId: video.snippet.channelId, }; } catch (e) { console.error("Error fetching video info:", e); return null; } }

function getChannelInfo(channelId) { try { var response = YouTube.Channels.list('snippet,statistics', { id: channelId, key: API_KEY }); if (response.items.length === 0) { return null; // 채널 정보가 없는 경우 null 반환 } var channel = response.items[0]; return { channelTitle: channel.snippet.title, subscriberCount: channel.statistics.subscriberCount, }; } catch (e) { console.error("Error fetching channel info:", e); return null; } }

10단계: 동영상 ID 추출 함수 작성

function extractVideoId(url) { if (!url) return null; var videoId; try { if (url.includes('youtu.be')) { videoId = url.split('youtu.be/')[1]; } else if (url.includes('youtube.com')) { const urlParams = new URLSearchParams(new URL(url).search); videoId = urlParams.get('v'); } return videoId; } catch (e) { console.error("Error extracting video ID:", e); return null; } }
