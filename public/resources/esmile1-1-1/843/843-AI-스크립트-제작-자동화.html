주제 : 유튜브 숏츠 콘텐츠 제작을 위한 AI 스크립트 작성 자동화

 

내용:

 

이 글은 잭(Jack)이라는 유튜버가 자신의 채널에서 공개한 영상 "이 AI 시스템은 숏츠 시청률을 100배 끌어올립니다(100% 자동화)"의 내용을 요약하고, 텍스트 형식의 30단계 가이드로 정리했습니다. 이 가이드는 사용자가 RSS 피드, 인공지능 리서치 에이전트, 스크립트 생성기 등 다양한 도구를 활용하여 유튜브 숏츠 콘텐츠 제작에 활용할 스크립트를 자동으로 생성하는 방법을 설명합니다.

 

사용 보안을 위해 개인 API 키 및 유료 서비스 가격 정보는 제외했습니다.

 

1. 머리말

 

이 가이드는 최신 인공지능 기술을 활용하여 유튜브 숏츠 콘텐츠 제작 효율을 높이는 방법을 설명합니다. 이 자동화 시스템을 사용하면 최신 뉴스 기사를 수집하고, 해당 기사의 요약, 핵심 정보, 중요성 등을 추출하여 콘텐츠 제작에 활용할 수 있습니다. 또한 자동 생성된 스크립트를 바탕으로 쉽게 영상을 촬영할 수 있습니다.

 

2. 도구 소개

 

이 자동화 시스템을 구동하기 위해 다음과 같은 도구가 필요합니다.

 

  • 메이크(Make): 무료 플랫폼으로 자동화 시스템 구축에 활용
  • RSS DoApp: 무료/유료 플랫폼으로 RSS 피드 관리
  • OpenAI GPT-3: 유료 인공지능 텍스트 생성 모델 (비용 절약 팁 참조)
  • Perplexity AI: 유료 인공지능 리서치 에이전트
  • Airtable: 무료/유료 플랫폼으로 데이터 정리 및 관리
  • 구글 문서: 스크립트 저장 및 편집

3. 비용 절약 팁

 

OpenAI GPT-3과 같은 유료 서비스는 다양한 모델과 가격대를 제공합니다. 처음 사용 시에는 저렴한 모델부터 시작하여 필요에 따라 업그레이드하는 것이 좋습니다. 또한 영상 제작에 반드시 최고 사양의 모델이 필요하지 않은 경우가 많습니다.

 

4. RSS 피드 설정

  • 메이크 플랫폼에 로그인합니다.
  • RSS DoApp으로 이동하여 회원가입합니다.
  • 관심 있는 주제(예: 인공지능)를 검색하여 RSS 피드를 생성합니다.
  • 생성된 RSS 피드 링크를 복사합니다.

5. RSS 피드 활용

  • 메이크 플랫폼에서 "Retrieve RSS Feed Items" 모듈을 추가합니다.
  • 복사한 RSS 피드 링크를 해당 모듈에 んでいます.
  • 최근 게시물 수(예: 2개)를 선택합니다.
  • (선택사항) 자동 실행 간격 설정 (예: 매일 1회)

6. 기사 내용 추출

  • "Get HTTP" 모듈을 추가하여 기사 본문을 가져옵니다.
  • 해당 모듈의 URL 필드에 RSS 피드에서 추출한 기사 링크를 입력합니다.
  • "HTML to Text" 모듈을 추가하여 HTML 코드를 텍스트로 변환합니다.

7. 인공지능 리서치

  • "Perplexity AI" 모듈을 추가합니다.
  • 텍스트 분석 모델을 선택합니다.
  • 기사 본문 텍스트를 해당 모듈에 입력합니다.
  • 분석 결과에서 핵심 토픽, 요약 정보, 연구 분석 내용 등을 추출합니다.

8. 스크립트 제작 에이전트

  • 세 개의 OpenAI GPT-3 에이전트를 생성합니다.
  • 첫 번째 에이전트는 "Virality Index"이며, 기사의 바이럴 지수(시청 가능성)를 생성합니다.
  • 두 번째 에이전트는 "Oneliner"이며, 기사 내용을 간략하게 요약합니다.
  • 세 번째 에이전트는 "Overview"이며, 기사의 전체적인 정보를 요약합니다.

9. 바이럴 지수 생성

  • "Virality Index

10. 바이럴 지수 생성

  • "Virality Index" 에이전트에 기사 본문과 인공지능 리서치 결과를 입력합니다.
  • 에이전트는 기사의 관련성, 독창성, 감정적 호소력 등을 평가하여 바이럴 지수를 산출합니다.

11. 간략한 요약 생성

  • "Oneliner" 에이전트에 기사 본문과 인공지능 리서치 결과를 입력합니다.
  • 에이전트는 기사의 핵심 내용을 50자 이내로 간략하게 요약합니다.

12. 상세 요약 생성

  • "Overview" 에이전트에 기사 본문과 인공지능 리서치 결과를 입력합니다.
  • 에이전트는 기사의 배경, 핵심 내용, 영향력 등을 상세하게 요약합니다.

13. 데이터베이스 생성

  • Airtable에 새로운 데이터베이스를 생성합니다.
  • 데이터베이스에 다음과 같은 필드를 추가합니다:
    • 기사 제목
    • 기사 링크
    • 간략 요약
    • 상세 요약
    • 바이럴 지수
    • 기사 작성 시간
    • 스크립트 생성 시간

14. 데이터 입력

  • 메이크 플랫폼에서 "Create Airtable Record" 모듈을 추가합니다.
  • 해당 모듈에 위에서 생성한 데이터를 입력합니다.

15. 스크립트 생성

  • 스크립트 생성 로직을 구현합니다. 예를 들어, 바이럴 지수가 높은 기사를 우선적으로 선택하여 스크립트를 생성합니다.
  • 스크립트 생성에 필요한 정보(제목, 요약, 핵심 키워드 등)를 데이터베이스에서 추출합니다.
  • OpenAI GPT-3를 활용하여 스크립트를 생성합니다.

16. 스크립트 저장

  • 생성된 스크립트를 구글 문서 또는 다른 클라우드 스토리지에 저장합니다.

17. 자동화 실행

  • 메이크 플랫폼에서 자동화를 실행합니다.
  • 설정한 시간 간격(예: 매일)에 따라 자동화가 실행되어 새로운 기사를 수집하고 스크립트를 생성합니다.

18. 영상 촬영

  • 생성된 스크립트를 활용하여 영상을 촬영합니다.
  • 영상 편집 소프트웨어를 사용하여 영상을 편집합니다.
  • 필요한 경우 음악, 효과음, 자막 등을 추가합니다.

19. 영상 업로드

  • 완성된 영상을 유튜브에 업로드합니다.
  • 영상 제목, 설명, 태그 등을 입력합니다.
  • 영상의 가시성을 높이기 위해 소셜 미디어 등을 활용하여 홍보합니다.

20. 성과 분석

  • 유튜브 애널리틱스를 활용하여 영상의 조회수, 시청 시간, 좋아요 수 등을 분석합니다.
  • 분석 결과를 바탕으로 향후 콘텐츠 제작 전략을 수정합니다.

21. 반복 및 개선

  • 자동화 시스템을 지속적으로 개선합니다.
  • 새로운 기능을 추가하거나 기존 기능을 최적화합니다.
  • 데이터베이스를 활용하여 콘텐츠 트렌드를 분석합니다.
  • 사용자 피드백을 수렴하여 콘텐츠 제작 방향을 조정합니다.

22. 추가적인 팁

  • 다양한 RSS 피드를 활용하여 다양한 주제의 콘텐츠를 생성합니다.
  • 인공지능 리서치 에이전트를 활용하여 심층적인 분석 결과를 얻습니다.
  • 스크립트 생성에 다양한 템플릿을 활용하여 다양한 스타일의 콘텐츠를 생성합니다.
  • 영상 편집에 효율적인 도구를 활용하여 시간을 절약합니다.
  • 유튜브의 알고리즘을 이해하고 최적화된 콘텐츠를 제작합니다.

이 가이드에서는 AI를 활용하여 유튜브 숏츠 콘텐츠 제작을 자동화하는 방법을 설명했습니다. 이러한 자동화 시스템을 통해 콘텐츠 제작 효율을 높이고, 더 많은 양질의 콘텐츠를 생산할 수 있습니다.
