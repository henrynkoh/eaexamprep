음성을 통한 AI 자동화에 대한 영상자료를 요약정리 하였습니다. 

 

소개

 

 

이 비디오에서는 음성만 사용하여 원하는 생각에서 바이럴 소셜 미디어 콘텐츠를 만드는 방법을 보여 드립니다. 생성한 모든 콘텐츠는 검토, 수정 및 게시물 승인에 필요한 관련 캡션 및 해시태그와 함께 쉽게 액세스할 수 있는 표에 저장됩니다. 게시 버튼을 누르면 콘텐츠가 자동으로 게시됩니다. 이 전체 프로세스는 음성 메모 후 완전히 자동화되며 LinkedIn, Facebook, Twitter, Instagram 등 원하는 플랫폼에 게시할 수 있습니다.

 

항상 그렇듯이 제가 제공하는 모든 자동화 기능은 아래 커뮤니티에서 확인하실 수 있습니다. '더보기'를 클릭하고 ' 가져오기'를 클릭한 다음 파일을 선택하고 '저장'을 클릭하기만 하면 자동으로 채워지고 커뮤니티에서 찾을 수 없는 독점 콘텐츠도 제공됩니다. 이제 정확히 어떻게 작동하는지, 설정하는 데 필요한 것, 실행 비용 등을 단계별로 함께 살펴보겠습니다. 이를 통해 여러분도 이와 동일한 작업을 수행하여 놀라운 자동화 기능을 활용할 수 있게 됩니다.

 

시작하기 전에

 

먼저 트리거(발발점)가 필요합니다. 이 비디오에서는 Slack 녹음 메모를 사용하지만 휴대폰의 음성 메모와 같이 오디오로 녹음된 모든 것이 트리거가 될 수 있습니다. 이 자동화에서는 전체 프로세스에서 단 한 문장도 입력할 필요가 없으므로 매우 간편합니다. 예를 들어 다음과 같이 오디오 클립을 녹음해 보겠습니다.

 

"저는 Dogecoin과 암호화폐의 미래에 대해 생각하고 있습니다. 어쩌면 12개월 이내에 바이럴 소셜 미디어 게시물이나 기사를 작성하고 싶습니다. Dogecoin과 앞으로의 암호화폐 시장 상황에 대해 이야기하고 싶어요."

 

1. 트리거 설정

 

녹음이 완료되면 '실행'을 한 번 누릅니다. Slack에서 파일을 찾았고 Whisper에 의해 음성이 텍스트로 변환되었습니다. 이제 화면에 표시된 대로 텍스트가 나타납니다.

 

2. 리서치 수행

 

다음으로 Perplexity AI를 사용하여 텍스트를 기반으로 최신 데이터와 통찰력을 얻습니다. 이를 통해 사용자의 생각과 최신 데이터를 결합하여 더욱 강력한 콘텐츠를 만들 수 있습니다.

 

3. 바이럴 게시물 생성

 

텍스트와 리서치 결과를 바탕으로 Perplexity AI는 특정 플랫폼(이 경우 Instagram)에 맞게 게시물을 만듭니다.

 

4. 이미지 생성

 

좋은 게시물에는 이미지가 필수적입니다. DALL-E 3은 텍스트 프롬프트를 사용하여 게시물 내용과 일치하는 이미지를 생성합니다.

 

5. Airtable 연결

 

마지막으로 Airtable이라는 스프레드시트 플랫폼과 연결하여 생성된 콘텐츠를 저장합니다. 이를 통해 게시물 승인 및 게시 프로세스를 쉽게 관리할 수 있습니다.

 

6. 게시 승인

 

Airtable에서 생성된 게시물을 검토하고 승인할 수 있습니다. 승인되면 게시물이 자동으로 선택한 플랫폼에 게시됩니다.

이제 30단계에 걸쳐 전체 프로세스를 자세히 살펴보겠습니다.

 

1. 트리거

  • 트리거는 음성 메모를 저장할 수 있는 플랫폼 선택 (Slack, 휴대폰 음성 메모 등)
  • Slack 메시지의 개인 채널 메시지 찾기
  • Slack에서 오디오 파일 다운로드

2. 음성 텍스트 변환

  • OpenAI의 Whisper 사용하여 오디오 파일 텍스트 변환

3. Perplexity AI 프롬프트 생성

  • GPT-3.5 turbo 모델 사용하여 텍스트 프롬프트 생성 (비용 절약)
  • 프롬프트 내용: Perplexity AI에게 사용자의 음성 메모 주제와 최신 5가지 사실/통찰력/업데이트를 찾도록 지시
  • *4. Per

5. Perplexity AI 검색

  • Perplexity AI의 MixoL 8X 7B 모델을 사용하여 생성된 프롬프트를 기반으로 웹 검색
  • 최신 뉴스 및 통찰력을 수집

6. 바이럴 게시물 생성

  • GPT-4 모델을 사용하여 Perplexity AI의 검색 결과와 음성 메모 텍스트를 기반으로 바이럴 게시물 생성
  • 게시물은 플랫폼별 최적화된 형식으로 작성됨 (Instagram, LinkedIn, Twitter 등)

7. 이미지 생성 프롬프트 생성

  • GPT-3.5 turbo 모델을 사용하여 이미지 생성 프롬프트 생성
  • 프롬프트는 게시물 내용과 일치하는 이미지 스타일 및 키워드 포함

8. 이미지 생성

  • DALL-E 3을 사용하여 생성된 프롬프트를 기반으로 이미지 생성

9. Airtable에 데이터 저장

  • 생성된 게시물과 이미지를 Airtable에 저장
  • Airtable은 데이터베이스 역할을 하며, 게시물 승인 및 게시를 위한 플랫폼으로 사용

10. 게시물 승인

  • Airtable에서 게시물을 검토하고 승인
  • 승인된 게시물은 자동으로 선택된 플랫폼에 게시

자동화의 장점

  • 효율성: 시간과 노력을 절약
  • 생산성: 더 많은 콘텐츠를 생성할 수 있음
  • 창의성: AI의 도움으로 새로운 아이디어와 콘텐츠를 생성할 수 있음
  • 일관성: 브랜드의 일관된 메시지를 유지할 수 있음

주의사항

  • AI는 완벽하지 않으므로 생성된 콘텐츠를 항상 검토할 필요가 있음
  • AI를 사용할 때는 윤리적 고려사항을 염두에 두어야 함
  • AI는 도구일 뿐이며, 창의적인 아이디어는 여전히 인간의 역할임

결론

 

이러한 AI 자동화를 통해 소셜 미디어 콘텐츠 생성을 간소화하고 효율화할 수 있습니다. 음성 메모를 통해 아이디어를 기록하고, AI가 그 아이디어를 발전시켜 고품질의 콘텐츠를 생성하는 프로세스를 자동화할 수 있습니다. 이를 통해 더 많은 시간을 창의적인 작업에 할애하고 비즈니스 성장에 집중할 수 있습니다.
