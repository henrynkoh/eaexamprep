GPT를 활용한 AI 비트코인 투자 자동화 구현하기

비트코인 투자에 인공지능을 활용하는 방법에 대해 알아보겠습니다. 최근 GPT와 같은 대규모 언어 모델의 발전으로 AI를 활용한 투자 자동화가 가능해졌습니다. 이 글에서는 GPT를 이용해 비트코인 자동 매매 시스템을 구축하는 방법을 단계별로 설명하겠습니다.

1. 개요

GPT를 활용한 AI 비트코인 투자 자동화는 다음과 같은 과정으로 이루어집니다:

  1. 비트코인 차트 데이터 수집
  2. GPT에 데이터 제공 및 투자 판단 요청
  3. GPT의 판단에 따른 자동 매매 실행

이를 통해 AI가 시장 상황을 분석하고 투자 결정을 내리는 자동화된 시스템을 구축할 수 있습니다.

2. 환경 설정

필요한 도구 설치

  • Python 3.9 이상
  • Visual Studio Code
  • 업비트 계정 및 API 키
  • OpenAI API 키

라이브러리 설치

pip install python-dotenv
pip install openai
pip install pyupbit

3. 데이터 수집

업비트 API를 통해 비트코인 차트 데이터를 수집합니다:

import pyupbit

df = pyupbit.get_ohlcv("KRW-BTC", interval="day", count=30)

4. GPT 활용

OpenAI API를 사용해 GPT에 데이터를 전달하고 투자 판단을 요청합니다:

import openai

openai.api_key = "YOUR_API_KEY"

response = openai.ChatCompletion.create(
  model="gpt-3.5-turbo",
  messages=[
    {"role": "system", "content": "You are a bitcoin investment expert."},
    {"role": "user", "content": f"Here's the 30-day bitcoin price data: {df.to_json()}. Should I buy, sell, or hold?"}
  ]
)

decision = response['choices'][0]['message']['content']

5. 자동 매매 실행

GPT의 판단에 따라 매수/매도를 실행합니다:

if "buy" in decision.lower():
    upbit.buy_market_order("KRW-BTC", krw * 0.9995)
elif "sell" in decision.lower():
    btc = upbit.get_balance("KRW-BTC")
    upbit.sell_market_order("KRW-BTC", btc)
