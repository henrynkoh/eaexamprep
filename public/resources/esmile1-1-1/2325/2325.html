주요 내용

  • Hostinger Horizons는 AI 기반의 도구로, 사용자가 대화를 통해 웹사이트나 애플리케이션을 생성하고 호스팅할 수 있습니다.
  • 이 도구는 코드 작성, 호스팅 설정, 실시간预览 기능을 제공하며, 사용자가 기술적인 지식 없이도 웹사이트를 만들 수 있도록 지원합니다.
  • 그러나 AI의 한계로 인해 간혹 오류가 발생할 수 있으며, 이를 해결하기 위해 추가적인 프롬프트가 필요할 수 있습니다.

사용 방법 (30단계)

  1. Hostinger 계정 생성: Hostinger에 가입하여 호스팅 플랜을 구매합니다.
  2. Hostinger Horizons 선택: 호스팅 플랜에서 Hostinger Horizons 도구를 선택합니다.
  3. 프로젝트 시작: 새로운 프로젝트를 시작하고, AI와 대화를 통해 웹사이트나 애플리케이션의 기능을 설명합니다.
  4. 기본 레이아웃 설정: AI가 기본 레이아웃을 생성합니다.
  5. 기능 추가: 예약 시스템, 이메일 전송 기능 등 필요한 기능을 추가합니다.
  6. 디자인 수정: 색상, 브랜드 이름 등을 입력하여 디자인을 수정합니다.
  7. UI/UX 개선: 최신 디자인 요소와 애니메이션 효과를 추가합니다.
  8. Accordion 메뉴 추가: FAQ 섹션에 Accordion 메뉴를 추가합니다.
  9. YouTube API 사용: YouTube API를 통해 비디오를 추가합니다.
  10. 오류 해결: AI가 발생시킨 오류를 해결하기 위해 추가 프롬프트를 사용합니다.
  11. 가격 계산기 추가: 새로운 페이지에 가격 계산기를 추가합니다.
  12. 기존 페이지 복구: AI가 변경한 기존 페이지를 복구합니다.
  13. 소규모 디자인 수정: 버튼 위치 조정, 텍스트 추가 등 소규모 디자인 수정을 수행합니다.
  14. 모바일 친화성 확인: 웹사이트가 모바일에서 잘 작동하는지 확인합니다.
  15. SSL 인증서 적용: SSL 인증서가 자동으로 적용되었는지 확인합니다.
  16. 프로젝트 배포: 완성된 프로젝트를 배포합니다.
  17. 사용자 테스트: 사용자들이 웹사이트를 사용하는 것을 테스트합니다.
  18. 피드백 수집: 사용자 피드백을 수집하여 개선 사항을 파악합니다.
  19. AI 오류 해결: AI가 발생시킨 오류를 해결하기 위한 추가 프롬프트를 사용합니다.
  20. 코드 접근 불가: 코드를 직접 수정할 수 없는 점을 이해합니다.
  21. 프롬프트 제한: 사용 가능한 프롬프트 수를 관리합니다.
  22. 프롬프트 비용 관리: 프롬프트 사용에 따른 비용을 관리합니다.
  23. Hostinger 지원 활용: Hostinger의 고객 지원을 활용하여 문제를 해결합니다.
  24. 프로젝트 확장: 추가 기능을 통해 프로젝트를 확장합니다.
  25. 다양한 프로젝트 생성: Hostinger Horizons를 통해 다양한 프로젝트를 생성합니다.
  26. QR 코드 생성기 만들기: QR 코드 생성기와 같은 간단한 애플리케이션을 만듭니다.
  27. 보드 게임 도구 만들기: 보드 게임을 위한 도구를 만들고 온라인으로 공유합니다.
  28. YouTube 썸네일 다운로드기 만들기: YouTube 썸네일을 다운로드할 수 있는 도구를 만듭니다.
  29. YouTube 비디오 다운로드기 만들기: YouTube 비디오를 다운로드할 수 있는 도구를 만듭니다.
  30. 프로젝트 유지 관리: 생성한 프로젝트를 지속적으로 관리하고 개선합니다.
