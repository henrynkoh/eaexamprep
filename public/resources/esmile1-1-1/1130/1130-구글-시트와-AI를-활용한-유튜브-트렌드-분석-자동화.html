구글 시트와 AI를 활용한 유튜브 트렌드 분석 자동화

안녕하세요, 조윤정의 AI 실험실입니다. 오늘은 유튜브 트렌드를 분석하고 참고하기 위한 정보 수집을 자동화하는 방법을 소개해드리겠습니다. 구글 시트와 AI를 활용하여 누구나 쉽게 만들 수 있는 강력한 도구를 함께 만들어보겠습니다.

왜 이 도구가 필요한가?

유튜브 크리에이터로서 트렌드를 파악하고 인사이트를 얻는 것은 매우 중요합니다. 하지만 기존의 도구들은 비용이 많이 들거나 사용에 제한이 있어 불편함이 있었습니다. 예를 들어, 주원규 님이 만든 뷰트랩은 매우 유용하지만 한 달에 두 번만 무료로 사용할 수 있고, 더 많이 사용하려면 비용이 많이 듭니다[1].

이러한 문제를 해결하기 위해 우리는 구글 시트와 AI를 활용하여 무료로 사용할 수 있는 강력한 도구를 만들어볼 것입니다. 이 도구를 사용하면 하루에 50번에서 100번까지 무료로 검색할 수 있어, 한 달에 약 100만 원 상당의 가치를 얻을 수 있습니다[1].

도구 제작 과정

1. 구글 시트 준비

먼저 구글 시트를 열고 새로운 스프레드시트를 만듭니다. 그리고 '확장 프로그램' 메뉴에서 '앱스 스크립트'를 선택합니다[1].

2. AI 활용하기

우리는 코드 작성을 위해 AI를 활용할 것입니다. 이번 예제에서는 Claude를 사용하지만, ChatGPT 등 다른 AI 도구를 사용해도 됩니다[1].

AI에게 다음과 같이 지시합니다: "당신은 구글 엔지니어 10년 차야. 유튜브와 구글 시트 앱스 스크립트 전문가야. 나는 유튜버인데 유튜브 트렌드를 분석하고 참고하기 위해서 정보를 수집하고 싶어. 구글 시트에 유튜브 정보를 불러오고 싶어. 검색어를 입력하면 그와 관련된 영상을 불러오게 해줘."[1]

3. API 키 설정

AI가 제공한 코드를 앱스 스크립트에 붙여넣고 실행합니다. 그러면 에러가 발생할 텐데, 이는 API 키가 필요하기 때문입니다. YouTube Data API v3 키를 발급받아 설정해주어야 합니다[1].

4. 코드 수정 및 기능 추가

AI의 도움을 받아 코드를 점진적으로 수정하고 기능을 추가합니다. 예를 들어, 검색 결과 수를 늘리거나 썸네일 이미지를 표시하는 등의 기능을 추가할 수 있습니다[1].

완성된 도구의 기능

  1. 검색어 입력: 사용자가 원하는 키워드를 입력할 수 있습니다.
  2. 대량의 결과 표시: 한 번에 200개까지의 검색 결과를 표시합니다.
  3. 상세 정보 제공: 영상 링크, 썸네일, 업로드 날짜, 설명 등의 정보를 한 눈에 볼 수 있습니다.
  4. 썸네일 이미지 표시: 적당한 크기로 썸네일을 표시하여 시각적 참고가 가능합니다[1].

도구 사용의 장점

  1. 비용 절감: 기존 유료 서비스 대비 큰 비용 절감 효과가 있습니다.
  2. 사용 제한 없음: 하루에 여러 번 검색이 가능합니다.
  3. 커스터마이징 가능: 필요에 따라 기능을 추가하거나 수정할 수 있습니다.
  4. 데이터 활용: 수집한 데이터를 다양한 방식으로 분석하고 활용할 수 있습니다[1].

주의사항

  1. API 사용량 제한: YouTube Data API에는 일일 할당량이 있으므로 과도한 사용에 주의해야 합니다.
  2. 데이터 보안: API 키 등의 중요 정보를 안전하게 관리해야 합니다.
  3. 저작권 준수: 수집한 정보를 활용할 때 저작권 관련 법규를 준수해야 합니다[1].

향후 발전 방향

  1. 더 많은 분석 기능 추가: 예를 들어, 인기 있는 태그 분석, 시청자 반응 분석 등을 추가할 수 있습니다.
  2. 자동화 수준 향상: 정기적인 데이터 수집 및 보고서 생성 등을 자동화할 수 있습니다.
  3. 다른 플랫폼 연동: 인스타그램, 틱톡 등 다른 소셜 미디어 플랫폼의 데이터도 함께 분석할 수 있도록 확장할 수 있습니다[1].

이 도구를 활용하면 유튜브 크리에이터로서 더 효과적으로 트렌드를 파악하고, 콘텐츠 전략을 수립하는 데 큰 도움이 될 것입니다. 여러분의 채널 성장에 이 도구가 기여하기를 바랍니다.

도구 사용 방법 20단계

  1. 구글 계정으로 로그인합니다.
  2. 구글 드라이브에서 새 스프레드시트를 만듭니다.
  3. 상단 메뉴에서 '확장 프로그램'을 클릭합니다.
  4. '앱스 스크립트'를 선택합니다.
  5. 앱스 스크립트 편집기가 열리면, 기존 코드를 모두 지웁니다.
  6. AI로부터 받은 코드를 붙여넣습니다.
  7. 코드를 저장합니다 (디스크 모양 아이콘 클릭).
  8. '실행' 버튼을 클릭합니다.
  9. 권한 요청 팝업이 뜨면 '검토'를 클릭합니다.
  10. 구글 계정을 선택하고 '고급'을 클릭한 후 '안전하지 않음으로 이동'을 선택합니다.
  11. '허용'을 클릭하여 스크립트에 필요한 권한을 부여합니다.
  12. 스프레드시트로 돌아가 페이지를 새로고침합니다.
  13. 상단에 새로운 메뉴 'YouTube 검색'이 생성되었는지 확인합니다.
  14. 'YouTube 검색' 메뉴를 클릭하고 'API 키 설정'을 선택합니다.
  15. 발급받은 YouTube Data API v3 키를 입력합니다.
  16. A1 셀에 검색하고 싶은 키워드를 입력합니다.
  17. 다시 'YouTube 검색' 메뉴를 클릭하고 '검색 실행'을 선택합니다.
  18. 검색 결과가 스프레드시트에 채워지는 것을 확인합니다.
  19. 썸네일 이미지를 보려면 '액세스 허용' 알림이 뜨면 '허용'을 클릭합니다.
  20. 결과를 분석하고 필요한 정보를 활용합니다.

이 20단계를 따라하면, 누구나 쉽게 이 강력한 유튜브 트렌드 분석 도구를 사용할 수 있습니다. 처음에는 조금 복잡해 보일 수 있지만, 한 번 설정해 놓으면 이후에는 매우 간단하게 사용할 수 있습니다. 이 도구를 통해 여러분의 유튜브 전략이 한층 더 발전하기를 바랍니다.
