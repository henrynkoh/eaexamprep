GPT 4 Sheet and Docs 확장 프로그램 소개

GPT 4 Sheet and Docs 확장 프로그램은 구글 시트에서 GPT를 활용할 수 있게 해주는 강력한 도구입니다. 이 확장 프로그램을 통해 다양한 GPT 함수를 사용하여 데이터 분석, 텍스트 처리, 자동화 작업 등을 수행할 수 있습니다.

확장 프로그램 설치 및 실행

  1. 구글 시트에 접속합니다. (주소창에 sheet.new 입력)
  2. 상단 메뉴에서 '확장 프로그램' 선택
  3. 'GPT for Sheets' 선택
  4. 'Open' 버튼을 눌러 GPT 4 Sheet 실행

GPT 4 Sheet 인터페이스 소개

  • GPT Function 목록: 사용 가능한 모든 GPT 함수 열거
  • 모델 선택: GPT-3.5, GPT-4 등 다양한 모델 선택 가능
  • 답변 특성 조절: Precise, Neutral, Creative 선택 또는 Temperature 값 조절
  • 최대 응답 크기 설정: 생성되는 답변의 토큰 수 제한

주요 GPT 함수 소개

1. GPT 함수

기본적인 GPT 요청을 수행합니다.

=GPT("프롬프트 내용")

2. GPT List 함수

여러 셀에 걸쳐 답변을 출력합니다.

=GPT_LIST("프롬프트 내용", 출력할 셀 수)

3. GPT Fill 함수

자동 채우기 기능을 제공합니다.

=GPT_FILL(참고할 셀 범위, 적용할 셀 범위)

4. GPT Classify 함수

텍스트를 카테고리로 분류합니다.

=GPT_CLASSIFY(분류할 내용, 카테고리 목록)

5. GPT Format 함수

데이터 형식을 변경합니다.

=GPT_FORMAT(변경할 셀, "원하는 형식")

6. GPT Extract 함수

텍스트에서 특정 정보를 추출합니다.

=GPT_EXTRACT(원본 텍스트, "추출할 정보")

데이터 분석 예시: 사용자 리뷰 분석

1. 지역 분포 분석

  1. 주소 데이터에서 도시 정보 추출
  2. GPT Fill 또는 GPT Extract 함수 사용
  3. 추출된 도시 정보로 차트 생성

2. 감정 분석

  1. GPT Classify 함수로 피드백 내용 분류 (긍정/부정/중립)
  2. 분류 결과를 차트로 시각화

3. 주제별 분류

  1. GPT Classify 함수로 피드백 내용을 주제별로 분류
  2. 분류 결과를 차트로 시각화

4. 부정적 피드백 분석

  1. 부정적 피드백만 필터링
  2. 주요 불만 사항 파악
  3. GPT 함수로 맞춤형 팔로업 이메일 작성
