가비아 도메인 등록 및 티스토리/블로그스팟 연동 가이드

1. 가비아에서 도메인 등록하기

가비아에서 도메인을 등록하려면 다음 단계를 따릅니다.

  1. 가비아 로그인
    • 가비아 계정 아이디와 비밀번호를 입력합니다.
    • 보안 문자를 확인한 후 [로그인]을 클릭합니다1.
  2. 도메인 검색
    • 홈페이지 상단 [도메인]에 마우스를 올린 후 [도메인 검색]을 선택합니다1.
    • 원하는 도메인명을 입력하고 [검색]을 클릭해 사용 가능 여부를 확인합니다.
  3. 상표권 등록 (필요 시)
    • 신규 gTLD 도메인의 경우, 상표권 보호 등록 서비스(TMCH)를 신청할 수 있습니다13.
    • 상표권 등록은 약 2~3주의 심사 기간이 소요되며, 일반 등록보다 비용이 추가될 수 있습니다3.
  4. 결제 및 등록 완료
    • 도메인 등록 비용을 결제하면 등록 절차가 완료됩니다1.

2. DNS 설정 및 티스토리/블로그스팟 연동

2.1. 티스토리 블로그에 도메인 연결

  1. 티스토리 설정
    • 티스토리 관리자 페이지에서 [블로그 설정] > [기본 설정]으로 이동합니다.
    • "도메인" 항목에 가비아에서 구매한 도메인을 입력합니다 (예: www.example.com).
  2. DNS 레코드 설정 (가비아)
    • 가비아 도메인 관리 페이지에서 DNS 설정을 합니다.
      • A 레코드: 티스토리 IP 주소 (예: 210.16.73.8)를 입력합니다.
      • CNAME: www를 dns.ticdn.com으로 연결합니다28.
    • 변경 사항이 적용되기까지 몇 분에서 몇 시간이 소요될 수 있습니다.
  3. HTTPS 설정
    • 티스토리에서 SSL 인증서를 발급받아 HTTPS를 활성화합니다.
    • 루트 도메인(example.com) 접속 시 www로 리디렉션되도록 설정합니다8.

2.2. 블로그스팟에 도메인 연결

  1. 블로그스팟 생성
    • 구글 블로그스팟에서 새 블로그를 만들고 기본 주소를 설정합니다6.
  2. 맞춤 도메인 설정
    • 블로그스팟 [설정] > [맞춤 도메인]에서 가비아 도메인을 입력합니다 (예: www.example.com)26.
    • www가 없는 도메인은 지원되지 않으므로 반드시 www를 포함합니다.
  3. DNS 레코드 설정 (가비아)
    • A 레코드: 구글 제공 IP 중 하나(216.239.32.21)를 입력합니다2.
    • CNAME:
      • www → ghs.google.com
      • 블로그스팟에서 제공하는 고유 CNAME (예: q3calwiayeuv)도 추가합니다26.
  4. HTTPS 및 애드센스 연동
    • 블로그스팟에서 [HTTPS 사용]을 활성화합니다.
    • 애드센스는 자동으로 연동되거나, 수동으로 ads.txt를 설정할 수 있습니다24.

3. 주의사항 및 문제 해결

  • 도메인 전파 시간: DNS 변경은 최대 48시간이 소요될 수 있습니다.
  • 티스토리 vs. 블로그스팟:
    • 티스토리는 애드센스 승인이 쉬우나 저품질 리스크가 있고, 블로그스팟은 무료이며 구글 생태계와 호환성이 좋습니다48.
  • 호스팅 업체 차이:
    • 카페24에서는 A 레코드를 1개만 입력할 수 있으므로, 가비아도 유사한 제한이 있을 수 있습니다2.

4. 마무리

가비아 도메인을 티스토리 또는 블로그스팟에 연결하면 기존 서버 없이도 블로그를 운영할 수 있습니다.
DNS 설정과 HTTPS 적용을 꼼꼼히 확인하고, 애드센스 연동으로 수익을 창출해 보세요.

가비아 도메인 등록 후 티스토리/블로그스팟 연동 가이드

1. DNS 레코드 설정 방법

가비아에서의 설정 절차

  1. A 레코드 설정
    • 티스토리: 210.16.73.8 IP 입력
    • 블로그스팟: 216.239.32.21, 216.239.34.21, 216.239.36.21, 216.239.38.21 IP 중 하나 선택12
    • 루트 도메인(example.com)은 호스트에 @ 입력, 서브도메인(www)은 www 입력1
  2. CNAME 레코드 설정
    • 티스토리: www → dns.ticdn.com1
    • 블로그스팟: www → ghs.google.com 및 구글에서 제공하는 고유 CNAME 추가2
  3. TXT 레코드 (필요 시)
    • 블로그스팟 도메인 소유권 확인용 TXT 레코드 추가2

2. 연동 시 주의사항

  • 도메인 전파 시간: 변경 사항 적용까지 최대 48시간 소요될 수 있음1
  • HTTPS 강제 적용: 티스토리/블로그스팟에서 SSL 인증서 발급 필요3
  • www 접두사 필수: 블로그스팟은 www 없이 루트 도메인 연결 불가2
  • 레코드 중복 방지: 동일 도메인으로 두 서비스 동시 연동 불가

3. 가비아 도메인 관리 방법

  1. DNS 설정 접근 경로
    • 로그인 → [My가비아] → [서비스 관리] → [DNS 관리툴]1
  2. 레코드 추가/수정
    • [+레코드 추가] 클릭 후 타입(A/CNAME/MX 등) 선택
    • 호스트 및 값 입력 시 **마침표(.)**로 끝나야 함(예: ghs.google.com.)1
  3. 최대 설정 가능 레코드: 200개1

4. DNS 오류 시 발생 문제

  1. 접속 장애
    • "DNS 서버 응답 없음" 오류 발생7
    • 웹사이트 로딩 지연 또는 연결 불가6
  2. 보안 문제
    • 잘못된 IP로 연결될 경우 피싱 사이트 유도 가능9
  3. 메일 서비스 중단
    • MX 레코드 오류 시 이메일 수발신 불가1
  4. SEO 영향
    • 검색 엔진이 사이트를 불안정한 것으로 판단하여 순위 하락3

5. 문제 해결 팁

  • 진단 도구 활용: nslookup 또는 dig로 DNS 확인7
  • 캐시 삭제: 브라우저 및 로컬 DNS 캐시 초기화7
  • 대체 DNS 사용: Google DNS(8.8.8.8) 또는 Cloudflare DNS(1.1.1.1)로 임시 변경7
  • 티스토리/블로그스팟 재검증: 설정 페이지에서 도메인 연결 상태 확인23

참고: DNS 변경 후 48시간 내 접속 불안정은 정상 현상이며, 전파 완료 후 재점검 필요16
