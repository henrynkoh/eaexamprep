구글 시트를 활용한 자동 문자 발송 시스템 구축하기

구글 시트와 앱스스크립트를 활용하여 자동으로 문자를 발송하는 시스템을 구축하는 방법에 대해 상세히 알아보겠습니다. 이 시스템은 업무 효율성을 크게 향상시킬 수 있는 강력한 도구입니다.

시스템 구축 준비

솔라피 API 키 준비

  • 솔라피 API 키는 문자 발송에 필수적입니다.
  • API 키 발급 방법은 별도의 5분짜리 영상을 통해 쉽게 얻을 수 있습니다[1].

구글 시트 설정

  1. 새로운 구글 시트 문서를 엽니다.
  2. 두 가지 주요 코드를 준비합니다:
    • 문자 발송용 시트 코드
    • 문자 발송 시트 설명용 구글 문서 코드[1]

앱스스크립트 설정

  1. 구글 시트에서 '확장 프로그램' > '앱스스크립트'로 이동합니다.
  2. 기존 코드를 모두 삭제하고 준비한 코드를 붙여넣습니다.
  3. API 키, 시크릿, 본인 전화번호를 코드 내 지정된 위치에 입력합니다[1].

권한 설정 및 실행

  1. 코드 저장 후 구글 시트로 돌아갑니다.
  2. 새로고침하면 'SMS 발송' 메뉴가 나타납니다.
  3. 'SMS 발송' > '시트 만들기' 클릭
  4. 권한 승인 과정을 거칩니다 (총 5번의 클릭 필요)[1].

문자 발송 테스트

  1. 생성된 창에 수신자 전화번호, 이름, 문자 내용을 입력합니다.
  2. 'SMS 발송'에서 '선택한 연락처로 SMS 발송' 클릭
  3. 발송 성공 메시지 확인 및 실제 수신 여부 체크[1]

사용자 가이드 생성

  1. 두 번째 코드를 앱스스크립트에 추가로 붙여넣습니다.
  2. 코드 저장 후 'createUserGuide' 함수 선택 및 실행
  3. 권한 승인 과정 반복
  4. 구글 드라이브의 '최근 문서함'에서 자동 생성된 설명서 확인[1]

시스템 활용 단계별 가이드

  1. 솔라피 API 키 발급받기
    • 솔라피 웹사이트 방문
    • 회원가입 및 로그인
    • API 키 발급 신청
    • 발급받은 API 키, 시크릿 키 안전하게 보관
  2. 새 구글 시트 문서 생성
    • 구글 드라이브 접속
    • '새로만들기' > '구글 시트' 선택
    • 새 시트에 적절한 이름 부여
  3. 앱스스크립트 열기
    • 구글 시트 상단 메뉴에서 '도구' > '스크립트 편집기' 선택
  4. 코드 복사 및 붙여넣기
    • 제공된 코드 전체 선택 (Ctrl+A)
    • 복사 (Ctrl+C)
    • 스크립트 편집기에 붙여넣기 (Ctrl+V)
  5. API 정보 입력
    • 코드 내 지정된 위치에 솔라피 API 키, 시크릿 키, 본인 전화번호 입력
  6. 코드 저장
    • 스크립트 편집기 상단의 저장 아이콘 클릭 또는 Ctrl+S
  7. 구글 시트로 돌아가기
    • 스크립트 편집기 탭 닫기
    • 구글 시트 새로고침
  8. 커스텀 메뉴 확인
    • 시트 상단에 'SMS 발송' 메뉴 생성 확인
  9. 시트 생성
    • 'SMS 발송' > '시트 만들기' 클릭
  10. 권한 승인
    • '계속' 버튼 클릭
    • Google 계정 선택
    • '고급' > '안전하지 않음(프로젝트 이름)으로 이동' 클릭
    • '허용' 버튼 클릭
  11. 테스트용 문자 정보 입력
    • 생성된 시트에 수신자 전화번호, 이름, 문자 내용 입력
  12. 문자 발송 테스트
    • 'SMS 발송' > '선택한 연락처로 SMS 발송' 클릭
  13. 발송 결과 확인
    • 팝업창의 성공 메시지 확인
    • 실제 수신 전화로 문자 도착 확인
  14. 사용자 가이드 코드 추가
    • 앱스스크립트로 다시 이동
    • 제공된 두 번째 코드 복사 및 붙여넣기
  15. 사용자 가이드 생성 함수 실행
    • 'createUserGuide' 함수 선택
    • '실행' 버튼 클릭
  16. 추가 권한 승인
    • 이전과 동일한 권한 승인 과정 반복
  17. 생성된 가이드 문서 확인
    • 구글 드라이브의 '최근 문서함' 확인
    • 자동 생성된 사용자 가이드 문서 열기
  18. 시스템 활용 및 테스트
    • 가이드 문서를 참고하여 시스템 기능 테스트
    • 다양한 시나리오로 문자 발송 시도
  19. 문제 해결 및 최적화
    • 발생 가능한 오류 상황 체크
    • 필요시 코드 수정 및 최적화
  20. 정기적인 시스템 점검
    • API 키 유효성 정기 확인
    • 새로운 기능 추가 고려
    • 사용자 피드백 반영 및 시스템 개선

이 자동화 시스템은 비즈니스 커뮤니케이션을 크게 개선할 수 있는 강력한 도구입니다. 단계별로 천천히 따라하면서 구축하시면, 효율적인 문자 발송 시스템을 손쉽게 만들 수 있습니다. 향후에는 이 시스템을 기반으로 더 복잡한 자동화 작업, 예를 들어 월별 정산서 PDF 생성 및 이메일 발송 등으로 확장할 수 있습니다[1]. 이러한 자동화 도구들을 활용하면 업무 생산성을 크게 향상시킬 수 있으며, 더 중요한 업무에 집중할 수 있는 여유를 가질 수 있습니다.
