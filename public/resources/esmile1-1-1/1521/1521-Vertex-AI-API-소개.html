Vertex AI API 소개

Vertex AI는 Google Cloud Platform에서 제공하는 종합적인 머신러닝 도구 및 서비스 모음입니다. 이 API를 통해 개발자와 데이터 과학자들은 AutoML이나 사용자 정의 코드를 사용하여 ML 모델을 구축하고 배포할 수 있습니다. 이 글에서는 Vertex AI API의 주요 기능과 사용 방법에 대해 자세히 알아보겠습니다.

Vertex AI의 주요 기능

Vertex AI는 다음과 같은 주요 기능을 제공합니다:

  1. 모델 개발: AutoML 또는 사용자 정의 모델 개발 지원
  2. 모델 학습: 대규모 분산 학습 환경 제공
  3. 모델 평가: 자동화된 모델 평가 및 성능 지표 제공
  4. 모델 배포: 확장 가능한 엔드포인트를 통한 모델 서빙
  5. 파이프라인: 머신러닝 워크플로우 자동화
  6. 데이터 관리: 데이터 레이블링 및 버전 관리

Vertex AI 시작하기

Vertex AI를 사용하기 위한 기본적인 단계는 다음과 같습니다:

  1. Google Cloud 프로젝트 생성
  2. Vertex AI API 활성화
  3. 서비스 계정 생성 및 권한 설정
  4. SDK 설치 및 초기화

Google Cloud 프로젝트 생성

먼저 Google Cloud Console에 로그인하여 새 프로젝트를 생성합니다. 모든 Google Cloud 서비스는 프로젝트 내에서 실행되므로 이 단계가 필수적입니다.

Vertex AI API 활성화

프로젝트를 생성한 후, Vertex AI API를 활성화해야 합니다. 이는 다음 명령어를 사용하여 수행할 수 있습니다:

gcloud services enable aiplatform.googleapis.com

또는 Google Cloud Console에서 직접 API를 활성화할 수도 있습니다.

서비스 계정 생성

Google Cloud의 다른 서비스에 액세스하려면 서비스 계정이 필요합니다. 서비스 계정을 생성하고 적절한 권한을 부여해야 합니다.

SDK 설치 및 초기화

Vertex AI를 사용하기 위해 Python SDK를 설치합니다:

pip install google-cloud-aiplatform

그런 다음 Python 코드에서 SDK를 초기화합니다:

from google.cloud import aiplatform

aiplatform.init(project='your-project-id', location='us-central1')

Vertex AI의 주요 구성 요소

모델 개발

Vertex AI는 AutoML과 커스텀 모델 두 가지 방식의 모델 개발을 지원합니다.

AutoML

AutoML을 사용하면 코딩 없이도 고품질의 ML 모델을 만들 수 있습니다. 지원되는 작업 유형은 다음과 같습니다:

  • 이미지 분류
  • 객체 감지
  • 자연어 처리
  • 표 데이터 분석

커스텀 모델

TensorFlow, PyTorch 등의 프레임워크를 사용하여 사용자 정의 모델을 개발할 수 있습니다. Vertex AI는 이러한 모델의 학습과 배포를 지원합니다.

모델 학습

Vertex AI는 강력한 분산 학습 환경을 제공합니다. GPU와 TPU를 포함한 다양한 하드웨어 옵션을 선택할 수 있습니다.

job = aiplatform.CustomTrainingJob(
    display_name="my-training-job",
    script_path="trainer/task.py",
    container_uri="gcr.io/cloud-aiplatform/training/tf-cpu.2-2:latest",
    requirements=["gcsfs==0.7.1"],
    model_serving_container_image_uri="gcr.io/cloud-aiplatform/prediction/tf2-cpu.2-2:latest",
)

model = job.run(
    dataset=dataset,
    model_display_name="my-trained-model",
    replica_count=1,
    machine_type="n1-standard-4",
    accelerator_type="NVIDIA_TESLA_K80",
    accelerator_count=1,
)

모델 평가

학습된 모델의 성능을 평가하기 위해 Vertex AI는 자동화된 평가 도구를 제공합니다.

model = aiplatform.Model('projects/my-project/locations/us-central1/models/{MODEL_ID}')
evaluation = model.get_model_evaluation()
eval_metrics = evaluation.metrics

모델 배포

학습된 모델은 Vertex AI 엔드포인트를 통해 쉽게 배포할 수 있습니다.

endpoint = aiplatform.Endpoint.create(display_name='my-endpoint')
model = aiplatform.Model('projects/my-project/locations/us-central1/models/{MODEL_ID}')
endpoint.deploy(
    model,
    min_replica_count=1,
    max_replica_count=5,
    machine_type='n1-standard-4',
    accelerator_type='NVIDIA_TESLA_K80',
    accelerator_count=1
)

배포된 모델에 대한 예측 요청은 다음과 같이 수행할 수 있습니다:

predictions = endpoint.predict(instances=[[6.7, 3.1, 4.7, 1.5], [4.6, 3.1, 1.5, 0.2]])

파이프라인

Vertex AI 파이프라인을 사용하면 전체 ML 워크플로우를 자동화할 수 있습니다. 데이터 전처리부터 모델 학습, 평가, 배포까지 모든 단계를 하나의 파이프라인으로 구성할 수 있습니다.

from google.cloud import aiplatform

def create_pipeline_job(
    project: str,
    location: str,
    pipeline_job_display_name: str,
    pipeline_job_template_path: str,
    parameter_values: Dict[str, Any],
) -> aiplatform.PipelineJob:
    pipeline_job = aiplatform.PipelineJob(
        display_name=pipeline_job_display_name,
        template_path=pipeline_job_template_path,
        parameter_values=parameter_values,
    )
    pipeline_job.submit()
    return pipeline_job

Vertex AI의 고급 기능

Gemini API

Vertex AI는 최신 생성형 AI 모델인 Gemini에 대한 액세스를 제공합니다. Gemini API를 사용하면 텍스트, 이미지, 오디오 등 다양한 형식의 입력을 처리할 수 있는 강력한 멀티모달 모델을 활용할 수 있습니다.

Gemini API를 사용하기 위해서는 Vertex AI in Firebase SDK를 사용해야 합니다. 이 SDK를 통해 모바일 또는 웹 앱에서 직접 Gemini API를 호출할 수 있습니다.

import { vertex } from '@ai-sdk/google-vertex';
import { generateText } from 'ai';

const { text, experimental_providerMetadata } = await generateText({
  model: vertex('gemini-1.5-pro', { useSearchGrounding: true }),
  prompt: 'List the top 5 San Francisco news from the past week. You must include the date of each article.'
});

보안 및 확장성

Vertex AI는 프로덕션 환경에서 사용할 수 있는 강력한 보안 기능을 제공합니다. Firebase App Check를 사용하여 인증되지 않은 클라이언트로부터 API를 보호할 수 있습니다.

또한, Cloud Firestore와 같은 Firebase 데이터베이스 서비스를 활용하여 구조화된 데이터를 관리하고, Firebase Remote Config를 사용하여 런타임 구성을 동적으로 설정할 수 있습니다.

Vertex AI 사용 사례

Vertex AI는 다양한 산업 분야에서 활용될 수 있습니다. 몇 가지 주요 사용 사례를 살펴보겠습니다:

  1. 금융 서비스: 사기 탐지, 신용 위험 평가, 주식 시장 예측
  2. 의료: 질병 진단, 의료 영상 분석, 개인화된 치료 계획
  3. 소매: 수요 예측, 고객 세분화, 추천 시스템
  4. 제조: 예측 유지보수, 품질 관리, 공급망 최적화
  5. 미디어 및 엔터테인먼트: 콘텐츠 추천, 사용자 행동 분석, 광고 타겟팅

Vertex AI의 장점

  1. 통합 플랫폼: 데이터 준비부터 모델 배포까지 전체 ML 라이프사이클을 하나의 플랫폼에서 관리
  2. 유연성: AutoML과 커스텀 모델 개발을 모두 지원
  3. 확장성: Google의 인프라를 활용한 대규모 학습 및 서빙 지원
  4. 최신 기술: Gemini와 같은 최신 AI 모델에 대한 액세스 제공
  5. 보안: 엔터프라이즈급 보안 및 규정 준수 기능

결론

Vertex AI는 머신러닝 프로젝트를 위한 강력하고 유연한 플랫폼입니다. AutoML을 통한 빠른 프로토타이핑부터 복잡한 커스텀 모델 개발까지, 다양한 수준의 ML 작업을 지원합니다. 또한 Gemini API와 같은 최신 기술에 대한 액세스를 제공하여 최첨단 AI 애플리케이션 개발을 가능하게 합니다.

 

Vertex AI를 사용하면 데이터 과학자와 개발자들은 인프라 관리보다는 모델 개발과 비즈니스 문제 해결에 더 집중할 수 있습니다. Google Cloud의 강력한 인프라와 결합된 Vertex AI의 기능은 기업이 AI 프로젝트를 빠르게 시작하고 확장할 수 있도록 돕습니다.

머신러닝과 AI가 비즈니스의 핵심 요소로 자리잡아가는 현재, Vertex AI는 기업이 이러한 기술을 효과적으로 활용할 수 있는 강력한 도구가 될 것입니다. 지속적으로 발전하는 이 플랫폼을 통해, 우리는 더욱 혁신적이고 지능적인 솔루션을 만들어낼 수 있을 것입니다.
