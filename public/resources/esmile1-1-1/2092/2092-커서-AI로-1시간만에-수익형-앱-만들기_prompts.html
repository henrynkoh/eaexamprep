소개

안녕하세요, 여러분! 오늘은 코딩 한 줄 없이 단 한 시간 만에 수익형 앱을 만드는 놀라운 방법을 소개해드리려고 합니다. 커서 AI(Cursor AI)라는 혁신적인 도구를 사용하여 이미지 파일 변환 기능과 서비스 소개 웹사이트를 뚝딱 만들어내는 과정을 함께 살펴보겠습니다1.

수익형 웹사이트의 잠재력

먼저, 이미지 확장자 변환 사이트인 'Convert.io'를 예로 들어보겠습니다. 이 사이트는 하루 방문자가 170만 명에 달하며, 광고만으로도 하루에 천만 원 이상의 수익을 올릴 수 있는 잠재력을 가지고 있습니다. 물론 이 사이트는 오랜 시간 동안 최적화되고 인지도를 쌓은 결과이지만, 간단하면서도 유용한 기능을 제공하는 웹사이트의 가치를 보여주는 좋은 예시입니다1.

커서 AI 소개

커서 AI는 최근 개발자들 사이에서 주목받고 있는 코드 에디터입니다. 일반 에디터와 비슷하지만, AI 기능이 통합되어 있어 개발 과정을 획기적으로 단축시킬 수 있습니다. 특히 '컴포저' 기능을 통해 한 번에 여러 파일을 자동으로 생성할 수 있어 개발 속도를 크게 높일 수 있습니다12.

커서 AI 설정 및 사용법

  1. 커서 AI 설치 및 14일 무료 프로 요금제 활성화
  2. 아마존 서버에 새 웹사이트 세팅 (선택사항)
  3. Remote SSH 확장 도구 설치 (서버 접속용)
  4. 커서 에디터에서 AI 실행 (Ctrl + I)
  5. 사용할 AI 모델 선택 (예: GPT-3.5)
  6. API 키 설정 (선택사항, 무료 버전 사용 시)
  7. 커서 Rules 파일 생성 및 설정
  8. 작업 환경 연결 (로컬 또는 서버)
  9. 무료 웹사이트 테마 설치 (선택사항)
  10. 컴포저 기능을 통해 웹사이트 생성 요청
  11. 에이전트 모드 활성화로 자동 파일 편집
  12. 생성된 코드 검토 및 승인
  13. 추가 기능 요청 (예: 이미지 변환)
  14. 오류 수정 및 기능 개선 요청
  15. 최종 결과물 확인 및 테스트

커서 AI를 활용한 웹사이트 개발 과정

1. 초기 설정

커서 AI를 설치하고 14일 동안 무료로 제공되는 프로 요금제를 활성화합니다. 무료 요금제도 있지만, 기능이 제한적이므로 프로 버전을 추천합니다1.

2. 작업 환경 준비

개인 아마존 서버에 새로운 웹사이트를 세팅하고, SSH 접속을 위해 Remote SSH 확장 도구를 설치합니다. 이렇게 하면 커서 AI가 직접 웹사이트에 파일을 올리고 처리하는 과정을 실시간으로 확인할 수 있습니다1.

3. AI 모델 선택

커서 AI 실행 시 사용할 모델을 지정합니다. 이는 매우 중요한 단계로, 모델의 성능에 따라 생성되는 코드의 품질이 결정됩니다. GPT-3.5, Claude, DeepSeek 등 다양한 모델 중에서 선택할 수 있습니다12.

4. 커서 Rules 설정

세팅 페이지의 'Rules for AI' 메뉴에서 서버 환경 정보를 입력합니다. PHP 버전, MySQL 버전, 설치된 PHP 모듈, 데이터베이스 접속 정보 등을 명시하면 AI가 이에 맞는 코드를 생성하여 오류를 최소화할 수 있습니다1.

5. 무료 테마 활용

웹사이트의 기본 디자인을 위해 무료 테마를 설치합니다. 모바일 최적화된 반응형 테마를 선택하면 더욱 좋습니다1.

6. 컴포저 기능 활용

커서 AI의 컴포저 기능을 사용하여 원하는 기능(예: 이미지 변환)과 웹사이트 제작을 요청합니다. 에이전트 모드를 활성화하면 AI가 자동으로 파일을 생성하고 편집합니다12.

7. 코드 검토 및 수정

AI가 생성한 코드를 검토하고 필요한 경우 수정을 요청합니다. 예를 들어, 메뉴가 누락되었다면 추가를 요청하고, 다중 파일 변환 기능이 필요하다면 구체적으로 지시합니다1.

8. 최종 확인 및 테스트

완성된 웹사이트의 기능을 테스트하고 필요한 경우 추가 수정을 진행합니다1.

커서 AI 사용 팁 및 주의사항

  1. 채팅 기능 활용: 컴포저 옆의 채팅 기능을 사용하여 AI에게 질문하고 조언을 구할 수 있습니다1.
  2. 모드 선택: 노멀 모드는 수동, 에이전트 모드는 자동으로 작업을 진행합니다. 신중한 작업이 필요할 때는 노멀 모드를 선택하세요1.
  3. AI의 한계 인식: AI가 때때로 실제로 존재하지 않는 요소를 참조하거나 잘못된 정보를 제공할 수 있습니다. 이런 경우 구체적인 지시와 파일 제공으로 해결할 수 있습니다1.
  4. 세션 관리: 컴포저 진행 내역이 많이 쌓이면 AI의 성능이 저하될 수 있습니다. 이럴 때는 커서를 재시작하면 됩니다1.
  5. 개발 지식의 중요성: 기본적인 개발 지식이 있으면 AI와의 소통이 더욱 원활해집니다. 하지만 초보자도 충분히 사용 가능합니다1.

결론

커서 AI를 활용하면 개발 기간을 획기적으로 단축할 수 있습니다. 코딩을 잘 모르는 분들도 AI의 도움을 받아 개발이 가능하며, 모르는 부분은 AI에게 1대1 과외를 받는다고 생각하면 됩니다. AI 기술의 빠른 발전으로 앞으로 개발 분야에 더 많은 혁신이 있을 것으로 예상됩니다1.

이제 여러분도 커서 AI를 활용하여 자신만의 수익형 앱을 만들어보세요. 창의적인 아이디어와 AI의 도움으로 새로운 가능성을 열어갈 수 있습니다!

이 30단계를 따라가면서 커서 AI를 활용하여 효율적으로 앱을 개발할 수 있습니다. 각 단계에서 AI의 도움을 받아 빠르게 진행하되, 중요한 결정은 개발자가 직접 내리는 것이 좋습니다. 또한, AI가 생성한 코드를 항상 검토하고 필요한 경우 수정을 요청하는 것이 중요합니다.

 

첨부된 파일에서 사용되거나 암시된 프롬프트들을 순서대로 정리하면 다음과 같습니다:

  1. 이미지를 드래그 앤 드롭으로 변환하는 기능 만들기
  2. 서비스 소개 웹사이트 만들기
  3. 무료 테마를 활용하여 웹사이트 만들기
  4. 무료 테마를 편집하여 웹사이트 만들기
  5. 이미지 변환 기능을 별도의 새 페이지에 만들기
  6. 상단 메뉴 추가하기
  7. 이미지 변환 오류 수정하기
  8. 두 개 이상의 이미지도 변환되도록 기능 개선하기

총 8개의 프롬프트가 사용되었거나 암시되었습니다. 제가 제공한 목록에는 첨부 파일에서 명시적으로 언급되거나 암시된 모든 프롬프트가 포함되어 있습니다. 이 이외에 설명자료에 포함된 내용은 다음과 같습니다.

 

사용자로부터 이미지 파일을 제공받아 이미지 확장자를 변환해주는 웹사이트 프로그램을 만들거야.

사용자는 단지 드래그엔드롭으로 임지 파일들을 웹페이지 특정 영역에 넣으면 해당 이미지를 다른 이미지 확장자로 변환 가능해야 해.
