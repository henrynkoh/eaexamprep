유튜브 트렌드 분석을 위한 구글 시트 자동화 도구 만들기

안녕하세요, 조윤정의 AI 실험실입니다. 오늘은 유튜브 트렌드를 분석하고 참고하기 위한 정보 수집 도구를 만드는 방법을 소개해드리겠습니다. 이 도구는 구글 시트와 앱스 스크립트를 활용하여 유튜브 API를 통해 원하는 키워드의 영상 정보를 자동으로 수집할 수 있게 해줍니다.

왜 이 도구가 필요한가?

유튜브 크리에이터로서 트렌드를 파악하고 경쟁 채널의 컨텐츠를 분석하는 것은 매우 중요합니다. 하지만 수동으로 이 작업을 하는 것은 시간이 많이 소요되고 비효율적입니다. 시중에 있는 유료 도구들은 비용이 많이 들어 부담스러울 수 있습니다. 예를 들어, 주원규 님이 만든 뷰트랩의 경우 한 달에 35만 9,800원이나 듭니다[1].

 

이에 비해 우리가 만들 도구는 무료로 사용할 수 있으며, 하루에 50번에서 100번까지 검색이 가능합니다. 이는 유료 서비스를 사용했을 때 1년에 1,200만 원 이상 들어갈 수 있는 비용을 절감할 수 있게 해줍니다[1].

도구의 주요 기능

  1. 키워드 검색: 원하는 키워드를 입력하면 관련된 유튜브 영상 정보를 수집합니다.
  2. 대량 데이터 수집: 한 번에 100개, 200개, 심지어 500개 이상의 영상 정보를 가져올 수 있습니다.
  3. 상세 정보 제공: 영상 링크, 썸네일, 제목, 설명, 조회수, 업로드 날짜 등 다양한 정보를 제공합니다.
  4. 시각화: 썸네일 이미지를 직접 시트에서 볼 수 있어 빠른 분석이 가능합니다.
  5. 사용자 친화적 인터페이스: 구글 시트 내에서 간단한 버튼 클릭으로 검색이 가능합니다.

도구 제작 과정

이제 이 강력한 도구를 만드는 과정을 단계별로 살펴보겠습니다.

1. 구글 시트 생성

먼저 새로운 구글 시트를 생성합니다. 구글 앱스에서 구글 시트를 선택하고 새 문서를 만듭니다[1].

2. 앱스 스크립트 열기

구글 시트에서 '확장 프로그램' 메뉴를 클릭하고 '앱스 스크립트'를 선택합니다. 이것이 우리의 자동화 도구를 만들 플랫폼입니다[1].

3. API 키 준비

유튜브 데이터 API를 사용하기 위해서는 API 키가 필요합니다. 이 과정은 별도의 영상을 통해 상세히 설명되어 있으며, 3-5분 정도면 누구나 쉽게 받을 수 있습니다[1].

4. 코드 작성

앱스 스크립트 편집기에 필요한 코드를 작성합니다. 이 과정에서 AI 도구(예: Claude)를 활용하여 코드를 생성하고 수정할 수 있습니다[1].

5. 코드 저장 및 실행

코드를 작성한 후 반드시 저장을 해야 합니다. 저장 후 실행 버튼을 눌러 코드를 실행합니다[1].

6. 권한 설정

처음 실행 시 권한 설정이 필요합니다. '권한 검토' 버튼을 클릭하고, 자신의 구글 계정을 선택한 후 '고급'을 클릭하여 '안전하지 않음'으로 표시된 프로젝트를 허용합니다[1].

7. 메뉴 생성 확인

코드가 성공적으로 실행되면 구글 시트에 새로운 메뉴 항목이 생성됩니다. 'YouTube 검색'이라는 메뉴가 추가되었는지 확인합니다[1].

8. API 키 입력

생성된 메뉴에서 API 키를 입력할 수 있는 옵션을 선택하고, 준비한 API 키를 입력합니다[1].

9. 검색어 입력 및 실행

검색어를 입력할 수 있는 칸에 원하는 키워드를 입력하고 '검색 실행' 버튼을 클릭합니다[1].

10. 결과 확인

검색이 완료되면 시트에 결과가 표시됩니다. 영상 제목, 링크, 썸네일, 조회수 등의 정보를 확인할 수 있습니다[1].

11. 결과 개수 조정

기본적으로 50개의 결과가 표시되지만, 코드를 수정하여 100개, 200개 등으로 늘릴 수 있습니다[1].

12. 썸네일 크기 조정

썸네일 이미지가 너무 작다면 코드를 수정하여 크기를 키울 수 있습니다[1].

13. 코드 버전 관리

코드를 수정할 때마다 새로운 버전으로 저장하여 관리하는 것이 좋습니다. 예를 들어, 'V1', 'V2' 등으로 구분할 수 있습니다[1].

14. 검색 상자 개선

팝업으로 나타나는 검색 상자 대신 고정된 검색 상자를 만들어 사용성을 높일 수 있습니다[1].

15. 에러 해결

코드 실행 중 에러가 발생하면 에러 메시지를 복사하여 AI 도구에 물어보거나, 구글 검색을 통해 해결 방법을 찾을 수 있습니다[1].

16. 결과 정렬 및 분석

수집된 데이터를 조회수, 구독자 수 등으로 정렬하여 인기 있는 영상이나 채널을 파악할 수 있습니다[1].

17. 데이터 활용

수집한 데이터를 바탕으로 트렌드를 분석하고, 새로운 영상 아이디어를 얻거나 경쟁 채널의 전략을 파악할 수 있습니다[1].

18. 정기적인 데이터 수집

주기적으로 같은 키워드로 검색을 실행하여 트렌드의 변화를 추적할 수 있습니다[1].

19. 다중 API 키 활용

검색 횟수를 늘리기 위해 여러 개의 API 키를 번갈아 사용하는 방식으로 코드를 개선할 수 있습니다[1].

20. 지속적인 개선

사용하면서 발견되는 불편한 점이나 추가로 필요한 기능들을 계속해서 코드에 반영하여 도구를 발전시킵니다[1].

결론

이렇게 만든 도구는 유튜브 크리에이터들에게 매우 유용한 자산이 될 수 있습니다. 트렌드 분석, 경쟁 채널 모니터링, 아이디어 발굴 등 다양한 목적으로 활용할 수 있으며, 무료로 사용할 수 있다는 큰 장점이 있습니다.

 

하지만 이 도구를 사용할 때는 몇 가지 주의할 점이 있습니다:

  1. API 사용량 제한을 주의해야 합니다. 하루에 너무 많은 검색을 하면 할당량을 초과할 수 있습니다[1].
  2. 수집한 데이터의 저작권을 존중해야 합니다. 다른 크리에이터의 아이디어를 그대로 복제하는 것은 바람직하지 않습니다.
  3. 데이터는 참고용으로만 사용하고, 최종적인 컨텐츠 기획과 제작은 자신만의 독창성을 발휘해야 합니다.

이 도구를 잘 활용하여 여러분의 유튜브 채널 성장에 도움이 되길 바랍니다. 앞으로도 AI와 자동화 도구를 활용한 다양한 팁들을 공유하도록 하겠습니다.
