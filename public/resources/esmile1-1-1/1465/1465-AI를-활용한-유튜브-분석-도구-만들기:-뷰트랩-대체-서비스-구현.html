AI를 활용한 유튜브 분석 도구 만들기: 뷰트랩 대체 서비스 구현

유튜브 크리에이터들에게 필수적인 도구인 뷰트랩(VidIQ)은 월 300만원에 달하는 고가의 서비스입니다. 하지만 AI와 구글 스프레드시트의 앱스 스크립트를 활용하면 이와 유사한 기능을 무료로 구현할 수 있습니다. 이 글에서는 AI 경영교수 이보의 영상을 바탕으로 이 과정을 상세히 설명하고, 실제 구현 방법을 단계별로 안내하겠습니다.

뷰트랩 소개

뷰트랩은 유튜브 영상을 분석해주는 서비스로, 다음과 같은 기능을 제공합니다:

  • 키워드 검색을 통한 영상 분석
  • 조회수, 구독자 수, 영상 수 등의 데이터 제공
  • 노출 확률, 기호도, 성과도 등의 지표 제공
  • 채널 분석 기능

뷰트랩의 가격 정책은 다음과 같습니다:

  • 무료 버전: 월 2회 사용 가능
  • 월 6만원: 10회 사용 가능
  • 월 15만원: 30회 사용 가능
  • 월 20만원: 100회 사용 가능
  • 월 300만원: 무제한 사용 가능

AI를 활용한 대체 서비스 구현

이제 AI와 구글 스프레드시트를 활용하여 뷰트랩과 유사한 기능을 구현하는 방법을 살펴보겠습니다. 이 과정은 크게 다음과 같은 단계로 나눌 수 있습니다:

  1. 구글 클라우드 콘솔에서 API 키 발급
  2. 앱스 스크립트 작성
  3. 유튜브 데이터 가져오기
  4. 데이터 분석 및 표시

구현 단계

  1. 구글 클라우드 콘솔(console.cloud.google.com)에 접속합니다.
  2. 새 프로젝트를 생성합니다 (예: "유튜브 애널리틱스").
  3. API 및 서비스 메뉴에서 "라이브러리"를 선택합니다.
  4. 검색창에 "YouTube Data API"를 입력하고 선택합니다.
  5. "사용" 버튼을 클릭하여 API를 활성화합니다.
  6. 왼쪽 메뉴에서 "사용자 인증 정보"를 선택합니다.
  7. "사용자 인증 정보 만들기" 버튼을 클릭하고 "API 키"를 선택합니다.
  8. 생성된 API 키를 안전한 곳에 복사해 둡니다.
  9. 구글 스프레드시트를 열고 "도구" 메뉴에서 "스크립트 편집기"를 선택합니다.
  10. AI에게 유튜브 데이터를 가져오는 앱스 스크립트 코드를 요청합니다.
  11. 생성된 코드를 스크립트 편집기에 붙여넣습니다.
  12. API 키를 코드의 적절한 위치에 입력합니다.
  13. 코드를 실행하여 유튜브 데이터를 가져옵니다.
  14. AI에게 추가 기능(예: 조회수, 구독자 수, 총 영상 수)을 요청합니다.
  15. 업데이트된 코드를 다시 스크립트 편집기에 붙여넣고 실행합니다.
  16. AI에게 성과도 계산 및 우선순위 매기기 기능을 요청합니다.
  17. 새로운 코드를 적용하고 실행하여 결과를 확인합니다.
  18. AI에게 메뉴 생성 기능을 요청하여 사용자 인터페이스를 개선합니다.
  19. 최종 코드를 적용하고 테스트합니다.
  20. 필요에 따라 추가 기능을 요청하고 개선합니다.

사용된 프롬프트

  1. "유튜브에서 영상에 대한 데이터를 구글 스프레드시트로 불러오는 앱스 스크립트를 만들고 싶어 만들어 줘"
  2. "유튜브 데이터 API 키를 어떻게 받을 수 있어?"
  3. "나는 채널을 내 채널 분석하는 게 아니라 키워드 입력하면 유튜브 검색 결과에 나온 유튜브 데이터를 구글 스프레드시트에 가져오고 싶어 앱스 스크립트 다시 만들어 줘"
  4. "좋아 잘했어 여기에 구독자수 대비 조회수가 높은 것을 성과도라고 할 건데 성과도에 대해서 네가 계산해서 우선순위를 매겨주는 걸 더해 줘"
  5. "됐습니다 너무 잘했어 근데 유튜브 콘텐츠에 대한 링크가 안 나오네 이것도 추가해 줘"
  6. "API 키값과 키워드 입력하는 거를 메뉴로 만들어서 메뉴를 클릭하면 나오게 해 줘"
  7. "좋았어 아주 잘하고 있어 아까 성과도 만들었는데 실행해 보니까 오래된 컨텐츠가 조회수가 많이 나오는 경향이 있더라고 그래서 성과도 이외에 최신 성과도라는 항목을 만들어서 컨텐츠를 업로드한 날짜도 계산해서 최근에 업로드한 것에 대비해서 성과도 높은 것을 계산해 줘"

구현 결과 및 장점

이렇게 구현된 서비스는 다음과 같은 정보를 제공합니다:

  • 영상 ID, 제목, 채널명, 설명, 게시일, 썸네일 URL
  • 조회수, 구독자 수, 총 영상 수
  • 성과도 (구독자 수 대비 조회수)
  • 최신 성과도 (업로드 날짜 고려)
  • 유튜브 링크

이 방식의 장점은 다음과 같습니다:

  1. 비용 절감: 월 300만원에 달하는 서비스를 무료로 구현할 수 있습니다.
  2. 커스터마이징: 필요한 기능을 직접 추가하고 수정할 수 있습니다.
  3. 학습 기회: AI와 코딩에 대한 이해를 높일 수 있습니다.
  4. 실시간 업데이트: AI의 도움으로 빠르게 기능을 개선할 수 있습니다.

주의사항 및 한계

  1. API 사용량 제한: 구글의 API 사용 정책을 확인하고 준수해야 합니다.
  2. 데이터 정확성: 공식 서비스에 비해 정확도가 떨어질 수 있습니다.
  3. 유지보수: 코드 업데이트와 오류 수정이 필요할 수 있습니다.
  4. 학습 곡선: 코딩과 API에 대한 기본적인 이해가 필요합니다.

결론

AI와 구글 스프레드시트를 활용하면 고가의 유튜브 분석 도구를 대체할 수 있는 서비스를 무료로 구현할 수 있습니다. 이 방법은 비용 절감뿐만 아니라 필요에 따라 기능을 커스터마이징할 수 있는 장점이 있습니다. 물론 공식 서비스에 비해 일부 한계가 있을 수 있지만, 기본적인 분석 기능을 무료로 사용할 수 있다는 점에서 매우 유용합니다.

 

이 프로젝트는 AI의 활용 가능성을 보여주는 좋은 예시입니다. 코딩 경험이 없는 사용자도 AI의 도움을 받아 복잡한 기능을 구현할 수 있습니다. 앞으로 AI를 활용한 다양한 업무 자동화와 도구 개발이 가능해질 것으로 기대됩니다.

 

마지막으로, 이런 방식의 개발은 지속적인 학습과 개선이 필요합니다. AI 기술과 API의 변화에 따라 코드를 업데이트해야 하며, 더 나은 분석 방법을 계속해서 연구해야 합니다. 하지만 이 과정 자체가 valuable한 학습 경험이 될 것입니다.

 

유튜브 크리에이터 여러분, 이제 고가의 분석 도구에 의존하지 않고도 자신만의 분석 도구를 만들어 활용해보세요. AI의 도움을 받아 여러분의 채널을 더욱 효과적으로 성장시킬 수 있을 것입니다.
