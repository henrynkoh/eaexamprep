Google Sheets의 Gemini 기반 향상된 스마트 채우기 기능 사용하기

Google Sheets에서 Gemini AI를 활용한 향상된 스마트 채우기 기능이 도입되었습니다. 이 기능은 수동으로 텍스트를 처리하는 작업을 더욱 쉽게 만들어줍니다. 불완전한 열 쌍 사이의 관계를 자동으로 감지하고 AI를 사용하여 나머지 값을 예측함으로써 즉시 채울 수 있게 해줍니다.

주요 특징

  • 자동 감지: 불완전한 열 쌍 사이의 관계를 자동으로 감지합니다.
  • AI 예측: Gemini AI를 사용하여 나머지 값을 예측합니다.
  • 즉시 채우기: 예측된 값을 즉시 채울 수 있습니다.
  • 데스크톱 지원: 현재 데스크톱 버전에서만 사용 가능합니다.

사용 조건

이 기능을 사용하기 위해서는 다음 중 하나의 조건을 충족해야 합니다:

  1. 적격한 Google Workspace 구독
  2. Google One AI Premium 구독
  3. Google Workspace Labs 조기 액세스 테스트 프로그램 참여

향상된 스마트 채우기 제안 확인 방법

향상된 스마트 채우기 제안은 두 가지 방식으로 나타납니다:

  1. 파일 편집 후: 파일에 편집이 이루어진 후 제안이 나타납니다.
  2. 실시간 제안: 수동으로 텍스트를 입력하는 동안 Sheets가 예측 가능한 패턴을 감지하면 즉시 제안이 나타납니다.

제안 처리 방법

향상된 스마트 채우기 제안이 나타나면 다음과 같이 처리할 수 있습니다:

  • 수락: 제안 버블을 클릭하여 수락합니다.
  • 거부: 확장된 제안 필에서 취소 아이콘을 클릭하여 해당 범위의 제안을 거부합니다.

사용 사례 예시

  1. 피드백을 주제별로 분류
  2. 뉴스 기사를 주제별로 정리
  3. 주소 데이터를 일관된 형식으로 변환
  4. 텍스트 필드에서 전화번호 추출

사용 팁

  • 향상된 스마트 채우기는 영어 값만 감지하고 지원합니다.
  • 감지를 트리거하려면 테이블에 최소 3개의 예시 행과 2개의 고유한 열 값이 있어야 합니다.
  • 텍스트 열에서만 트리거되며 숫자 및 날짜 열에서는 작동하지 않습니다.

피드백 제공

Gemini for Google Workspace는 지속적으로 학습 중이며 모든 요청을 지원하지 못할 수 있습니다. 부정확하거나 안전하지 않다고 생각되는 제안을 받았다면 피드백을 제출할 수 있습니다.

피드백 제출 방법

  1. 제안의 점 세 개 메뉴를 클릭합니다.
  2. '피드백 보내기'를 클릭하여 피드백을 제출합니다.

Gemini 기능 제안에 대한 이해

  • Gemini 기능 제안은 Google의 견해를 대표하지 않으며 Google에 귀속되어서는 안 됩니다.
  • 의료, 법률, 재정 또는 기타 전문적인 조언으로 Gemini 기능에 의존해서는 안 됩니다.
  • Gemini 기능은 부정확하거나 부적절한 정보를 제안할 수 있습니다. 사용자의 피드백은 Gemini를 더욱 유용하고 안전하게 만드는 데 도움이 됩니다.

향상된 스마트 채우기 기능 사용 방법 30단계 가이드

  1. Google Sheets를 엽니다.
  2. 새 스프레드시트를 만들거나 기존 스프레드시트를 엽니다.
  3. 데이터를 입력할 열을 선택합니다.
  4. 첫 번째 열에 예시 데이터를 최소 3행 입력합니다.
  5. 두 번째 열에 해당하는 데이터를 입력합니다.
  6. 두 열 사이의 관계를 명확히 하는 패턴을 만듭니다.
  7. 세 번째 행 이후부터 첫 번째 열에만 데이터를 계속 입력합니다.
  8. 두 번째 열의 빈 셀에 커서를 놓습니다.
  9. 향상된 스마트 채우기 제안이 나타나기를 기다립니다.
  10. 제안이 나타나면 제안 버블을 확인합니다.
  11. 제안이 적절하다고 판단되면 제안 버블을 클릭하여 수락합니다.
  12. 제안이 부적절하다고 판단되면 취소 아이콘을 클릭하여 거부합니다.
  13. 수락한 제안이 자동으로 채워지는 것을 확인합니다.
  14. 필요한 경우 수동으로 일부 셀의 내용을 수정합니다.
  15. 다른 열에 대해서도 같은 과정을 반복합니다.
  16. 전체 스프레드시트에 대해 일관성 있게 데이터가 채워졌는지 확인합니다.
  17. 특정 패턴이 반복되는 경우, 해당 패턴을 활용하여 더 많은 데이터를 자동으로 채웁니다.
  18. 데이터 형식이 일관되게 유지되는지 확인합니다.
  19. 오류가 있는 경우 수동으로 수정합니다.
  20. 대량의 데이터를 처리해야 하는 경우, 샘플 데이터로 패턴을 학습시킵니다.
  21. 학습된 패턴을 기반으로 나머지 데이터를 자동으로 채웁니다.
  22. 자동 채우기 결과를 검토하고 필요한 경우 수정합니다.
  23. 데이터 분석이 필요한 경우, 채워진 데이터를 기반으로 차트나 피벗 테이블을 만듭니다.
  24. 결과물을 동료들과 공유하고 피드백을 받습니다.
  25. 받은 피드백을 바탕으로 데이터를 추가로 정제합니다.
  26. 향상된 스마트 채우기 기능에 대한 피드백이 있다면 Google에 제출합니다.
  27. 기능 개선 사항이 있다면 '도움말' 메뉴를 통해 제안합니다.
  28. 정기적으로 Google Workspace 업데이트를 확인하여 새로운 기능을 파악합니다.
  29. 다른 Google Sheets 사용자들과 향상된 스마트 채우기 사용 팁을 공유합니다.
  30. 지속적으로 기능을 활용하며 작업 효율성을 높입니다.

이 가이드를 따라 Google Sheets의 향상된 스마트 채우기 기능을 효과적으로 활용할 수 있습니다. 이 기능은 데이터 입력 및 처리 작업을 크게 간소화하여 업무 생산성을 향상시킬 수 있습니다.
